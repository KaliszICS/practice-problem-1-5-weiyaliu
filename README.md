# Instructions  

Practice Problems 1.5

1. Create a character variable with the value 'a'.  Output it.

2. Ask the user to "Input a word: ".  Select the 3rd letter of the word and output it.  Remember to use the 3rd letter, not index 3.

3. Ask the user to "Input a word: ".  Output "The second character user entered was: " followed by the second character of the word they entered.

4. Ask the user to "Input a number: ", save it as an integer.  Output "Your number plus 1 is: " Followed by their number increased by one.  Do not alter the variable the input was saved in.

5. Ask the user to "Input a letter: ", save it as a string, then save it as a character.
	 Ask the user to "Input another letter: ", save it as a string, then save it as a character.
	 Output the result of the characters _added_ together (not concatenated).
